# Collision & Score 模組任務

## 目標

判斷跑酷階段的碰撞與越過士兵事件，並管理本局分數。

## 最小可執行任務

- [x] 建立婦女與士兵的碰撞檢查方法。
- [x] 建立低位士兵的碰撞規則。
- [x] 建立高舉長矛士兵的碰撞規則。
- [x] 判斷低位士兵是否已被跳過。
- [x] 判斷高舉長矛士兵是否已被蹲下避開。
- [x] 為每名士兵建立是否已計分的標記。
- [x] 每成功越過一名士兵增加 10 分。
- [x] 分數改變時觸發 `scoreChanged`。
- [x] 撞到士兵時觸發 `hitSoldier`。
- [x] 撞到士兵時停止後續加分。
- [x] 確認不保存最高分。
- [x] 確認不顯示最高分。

## 完成標準

- [x] 成功越過士兵會增加本局分數。
- [x] 撞到士兵會立即進入遊戲結束流程。
- [x] 每名士兵最多只加分一次。
