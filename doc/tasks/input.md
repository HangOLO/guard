# Input 模組任務

## 目標

統一處理 iPad 觸控與桌面鍵盤輸入，並轉換成場景可理解的抽象動作。

## 最小可執行任務

- [x] 綁定開始遊戲按鈕輸入。
- [x] 綁定重新開始按鈕輸入。
- [x] 綁定聲音開關按鈕輸入。
- [x] 綁定左右移動按鈕輸入。
- [x] 綁定跳躍按鈕輸入。
- [x] 綁定蹲下按鈕輸入。
- [x] 建立桌面鍵盤對應，支援左右、跳躍、蹲下、開始或重新開始。
- [x] 將實際輸入轉換為 `moveLeft`、`moveRight`、`jump`、`duck`、`start`、`restart`、`toggleSound`。
- [x] 支援左右移動的按住與放開。
- [x] 支援蹲下的按住與放開。
- [x] 防止同一次點擊重複觸發多次開始或重新開始。

## 完成標準

- [x] iPad 觸控按鈕能輸出正確抽象動作。
- [x] 桌面鍵盤能輸出正確抽象動作。
- [x] Input 模組不直接修改遊戲狀態或場景資料。
