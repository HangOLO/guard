# Dialogue 模組任務

## 目標

顯示接觸士兵後的 2-3 句短對話，並在對話完成後進入跑酷階段。

## 最小可執行任務

- [x] 建立對話資料清單。
- [x] 使用需求文件指定的短對話內容。
- [x] 建立對話初始化方法。
- [x] 建立顯示目前句子的資料欄位。
- [x] 建立前進到下一句的方法。
- [x] 在最後一句完成後觸發 `dialogueComplete`。
- [x] 將目前對話文字交給 UI/HUD 顯示。
- [x] 確認對話內容不顯示經文章節引用。
- [x] 對話完成後不再接收左右移動輸入。

## 完成標準

- [x] 接觸士兵後能看到短對話。
- [x] 對話完成後能進入跑酷階段。
- [x] 對話內容符合「守衛嚴密、不能靠近墓口」的情境。
