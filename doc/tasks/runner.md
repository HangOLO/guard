# Runner 模組任務

## 目標

實作橫向跑酷主玩法，包含婦女跳躍、蹲下、士兵障礙生成與階段性加速。

## 最小可執行任務

- [x] 建立跑酷階段初始化方法。
- [x] 建立婦女跑酷狀態：地面、跳躍、蹲下。
- [x] 實作 `jump` 動作，讓婦女跳起並落回地面。
- [x] 實作 `duck` 動作，讓婦女在按住期間保持蹲下。
- [x] 建立士兵障礙列表。
- [x] 生成低位士兵障礙。
- [x] 生成高舉長矛士兵障礙。
- [x] 讓士兵障礙由右向左移動。
- [x] 移除已離開畫面的士兵障礙。
- [x] 記錄跑酷已經過時間。
- [x] 每 15 秒提升一次速度或生成壓力。
- [x] 前 15 秒保持可學習的初始難度。
- [x] 每一幀將婦女與士兵資料交給 Collision & Score 檢查。

## 完成標準

- [x] 玩家能在跑酷階段跳躍與蹲下。
- [x] 低位士兵與高舉長矛士兵會持續出現。
- [x] 遊戲壓力會每 15 秒提升。
- [x] Runner 不直接更新分數或顯示結束畫面。
