# Game State 模組任務

## 目標

集中管理遊戲狀態、本局分數與聲音開關，作為流程狀態的唯一來源。

## 最小可執行任務

- [x] 定義狀態常量：`title`、`approach`、`dialogue`、`runner`、`gameOver`。
- [x] 建立目前狀態資料欄位。
- [x] 建立本局分數資料欄位，初始值為 `0`。
- [x] 建立聲音開關資料欄位。
- [x] 建立狀態切換方法。
- [x] 限制狀態只能依照設計流程切換。
- [x] 建立本局重設方法，重設分數與場景資料。
- [x] 確認不建立最高分保存欄位。
- [x] 提供 UI/HUD 可讀取的目前狀態與本局分數。

## 完成標準

- [x] 狀態能按 `title → approach → dialogue → runner → gameOver` 流程切換。
- [x] `restart` 後能重新開始本局。
- [x] Game State 不包含登入、後端、排行榜或最高分保存邏輯。
