# Asset 模組任務

## 目標

集中管理遊戲圖片、角色、背景、士兵、UI 與聲音資產的鍵名與取得方式。

## 最小可執行任務

- [x] 建立資產 manifest。
- [x] 定義婦女角色資產鍵名。
- [x] 定義低位士兵資產鍵名。
- [x] 定義高舉長矛士兵資產鍵名。
- [x] 定義墓口與石壁場景資產鍵名。
- [x] 定義跑酷背景資產鍵名。
- [x] 定義 UI 按鈕與面板資產鍵名。
- [x] 定義聲音開關圖示資產鍵名。
- [x] 定義背景音樂資產鍵名。
- [x] 定義跳躍、蹲下、加分、撞到士兵音效鍵名。
- [x] 提供依鍵名取得資產的方法。
- [x] 避免其他模組直接依賴實際檔案路徑。
- [x] 確認使用者提供的 4 張圖片只作風格參考，不作正式素材依賴。

## 完成標準

- [x] 其他模組可透過 Asset 模組取得資產。
- [x] 資產鍵名穩定且可讀。
- [x] 不把參考圖當作正式遊戲素材。
