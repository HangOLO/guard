# Approach Scene 模組任務

## 目標

實作婦女左右移動並接觸羅馬士兵的前置遊戲階段。

## 最小可執行任務

- [x] 建立接觸士兵階段的初始化方法。
- [x] 設定婦女角色初始位置。
- [x] 設定可接觸羅馬士兵的位置。
- [x] 限制婦女只能左右移動。
- [x] 根據 `moveLeft` 與 `moveRight` 更新婦女位置。
- [x] 設定左右移動邊界，避免角色離開畫面。
- [x] 建立婦女與士兵的接觸判定。
- [x] 接觸成功時只觸發一次 `contactSoldier` 事件。
- [x] 接觸成功後停止接觸士兵階段的移動更新。

## 完成標準

- [x] 玩家能用左右按鈕控制婦女靠近士兵。
- [x] 婦女接觸士兵後會進入對話流程。
- [x] 此階段不支援上下移動或自由探索。
