# Audio 模組任務

## 目標

統一管理背景音樂與音效，並支援 iPad 瀏覽器互動後播放限制。

## 最小可執行任務

- [x] 登記背景音樂資產。
- [x] 登記跳躍音效資產。
- [x] 登記蹲下音效資產。
- [x] 登記加分音效資產。
- [x] 登記撞到士兵音效資產。
- [x] 建立聲音解鎖流程，必須由玩家互動後啟動。
- [x] 建立背景音樂播放方法。
- [x] 建立背景音樂停止或暫停方法。
- [x] 建立音效播放方法。
- [x] 實作 `toggleSound`，同時控制背景音樂與音效。
- [x] 靜音時不播放背景音樂。
- [x] 靜音時不播放音效。
- [x] 在跳躍、蹲下、加分、撞到士兵事件時播放對應音效。

## 完成標準

- [x] 玩家互動後可以啟動背景音樂。
- [x] 聲音開關能控制播放與靜音。
- [x] 音效不分散寫在其他模組內。
